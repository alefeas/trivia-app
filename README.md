# Trivia App 🎡

Trivia App es un juego que consiste en una batalla de trivia entre 2 jugadores. El mismo fue desarrollado con [**React JS**](https://reactjs.org/).

- [**SASS**](https://sass-lang.com/install)

## Librerías 📚
